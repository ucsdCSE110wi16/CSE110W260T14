User Stories

1. As a user,
   I want to create a profile
   so that other users and I can see my profile.
   Priority: 10

2. As a user, 
   I want to be able to log in 
   so that I can access my profile. 
   Priority: 20

3. As a user,
   I can input my enrolled classes
   so that other users and I can see my classes schedule. 
   Priority: 30
 
4. As a user,
   I can input available study times, 
   so that other users and I can see my study times. 
   Priority: 40
 
5. As a user,
   I want to input dedicated study times
   So that other users can see when im available.
   Priority: 50

6. As a user,
   I want to be able to search for study partners
   So that we can study together.
   Priority: 60
 
7. As a user,
   I want to invite another user to my study session
   So that we can all study together
   Priority: 70

8. As a user,
   I want to accept an initation from another user
   So that I can study with them.
   Priority: 80

9. As a user,
   I want to create a study group
   So that I can study with others.
   Priority: 90
 
10. As a user, 
   I want to be able to send invites to my group 
   so that I can work with the people I want to.
   Priority: 100
 
11. As a user, 
   I want an inbox for group invites 
   so that I can see who wants me to be a part of their group.
   Priority: 110
 
12. As a user, 
   I want to invite people in my group to study 
   so that I can study on my own schedule.
   Priority: 120
 
16. As a user,
   I want to search for a specific user
   So that I can find them quicker
   Priority: 160

17. As a user,
   I want to see what groups im a part of
   So that I know which ones im part of
   Priority: 170
 
18. As a user,
   I want to block specific users
   So that they can't access my profile or send me invites.
   Priority: 180
 
19. As a user,
   I want to have advanced controls of my group
   So that I can change any of the aspects of the group.
   Priority: 190
 
20. As a user,
   I want to indicate the need for a lab partner
   So that other users who need a partner can find me.
   Priority: 200
 
21. As a User,
    I want to use a an interactive map
    So that I can see where people are studying
    Priority: 210
 
22. As a User,
    I want to search for study partners
    So that I can see who wants to study in the future.
    Priority: 220
 
23. As a User,
    I want to use the map feature to create a study session
    So that other users can see where I'm currently studying at.
    Priority: 230
